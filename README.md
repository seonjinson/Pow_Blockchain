Simple Pow BlockChain 
======================

The structure and function of simple blockchain based on pow method.

## Complier
That may make an error if the gcc version is different.
```

* Compiler
  - g++(gcc version 8.1.0)

```

## Implemantation on mac/linux

* compile & build
```

$ cd Pow_Blockchain
$ make
$ ./a_out

```

* clean
```

$ clean

```
